S1-1 · Activations exist for a reason

Linear boundary vs one ReLU hidden layer

A single linear layer can only separate space with a line. A ReLU hidden layer can compose many half-planes into a curved-looking boundary.

The money shot appears after training: the linear model remains a line; ReLU wraps the ring.

S1-2 · Depth without nonlinearity is a lie

Five linear layers still collapse to one line

Stacking affine maps without activation creates another affine map. Insert ReLUs between the same depth and the boundary can bend.

More data closes the train/test gap
